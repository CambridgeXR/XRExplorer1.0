## Requirements

### Works on:
- **Meta Quest Browser**
- **Android Chrome** (WebXR enabled)
- **iOS via WebXR Viewer**
  - Install from App Store
  - Open:
    ```
    https://cambridgexr.github.io/XRExplorer1.0/
    ```
  - Or use deep link:
    ```
    wxrv://https://cambridgexr.github.io/XRExplorer1.0/
    ```

### Does *not* work on:
- iOS Safari (no WebXR support)
- Desktop Chrome/Firefox/Safari without VR hardware
